# Anagram Detector Challenge

Check if two strings are anagrams.

## Problem

Given two strings, determine if they are anagrams of each other. An anagram uses the same letters in a different order.

## Examples

- `"listen"`, `"silent"` → `true`
- `"hello"`, `"world"` → `false`
- `"Dormitory"`, `"Dirty room"` → `true`
- `"", ""` → `true`

## Notes

- Ignore case
- Consider only alphanumeric characters
